# 🚀 15 Days of LeetCode Challenge  
**University DSA Learning Initiative (GDG Society)**

This repository documents my successful completion of the **15-Days-Of-LeetCode-Challenge**, an intensive problem-solving initiative conducted through my university under the **GDG Society**.

The goal of this challenge was to **build strong foundations in Data Structures & Algorithms (DSA)** by solving real interview-level problems on **LeetCode**, with a strict focus on **consistency, correctness, and clean implementation**.

---

## 🎯 Challenge Objectives

- Develop **problem-solving discipline** by solving problems daily  
- Master **core data structures and algorithmic patterns**  
- Improve **code readability, optimization, and logical thinking**  
- Translate **theory into practical, interview-ready solutions**

---
